IP Camera

IP cameras stands for Internal Protocal cameras. These are the cameras that uses internet to transmit video footage and if internet is not available, then it stores in temporary memory until internet is again reconnected. The cameras can be connected via LAN or Wi-Fi. The cables used in the IP cameras are ethernet cables that helps to transmit video from one device to another. There are some features like:-

  • Remote access
  • Resolution
  • Alerts
  • Connectivity
